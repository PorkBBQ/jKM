Commands
         help
                 * help
幫助指令
                 * db.help()
db層級help
                 * db.co1.help()
collection層級help
         show
                 * show dbs
列出database
                 * show collections
列出目前db下的collection
         use
                 * use db1
連接 db1
         db
                 * db
顯示目前db
         insert
                 * db.co1.insert({'a':1})
插入資料
         find
                 * db.co1.find()
查詢collection co1內的所有紀錄
                 * db.co1.find({'a':1}, {'_id':0})
查詢符合{'a':1}的紀錄, 不顯示_id欄位
         findOne
                 * db.co1.findOne()
隨機查詢一筆紀錄
         update
                 replace doc
                         * db.co1.update({'a':1}, doc)
符合{'a':1}的紀錄以doc替代
                 $set
                         * db.co1.update({'a':1}, {'$set:{'a':2}})
將符合{'a':1}的紀錄修改為{'a':2}
                 $unset
                         * db.co1.update({'a':1}, {'$unset:{'a':1}})
將符合{'a':1}的紀錄移除'a'鍵值
                 $inc
                         * db.co1.update({'a':1}, {'$inc:{'a':1}})
將符合{'a':1}的紀錄值+1
                 $push
                         * db.co1.update({'a':1}, {'$push:{'b':{'b1':1}}})
將符合{'a':1}的紀錄增加陣列b的元素
         remove
                 * db.co1.remove({'a':1})
刪除符合{'a':1}的紀錄
                 * db.co1.remove()
清空collection co1, 不刪除索引
         drop
                 * db.co1.drop()
刪除collection co1及索引
