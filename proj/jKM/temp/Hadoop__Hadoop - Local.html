         redhat: version:6.2

path
         hadoop: /opt/hadoop-1.2.1
         jdk: /opt/jdk1.7.0_45
process
         download
                 * wget http://ftp.tc.edu.tw/pub/Apache/hadoop/core/stable/hadoop-1.2.1.tar.gz /opt
下載 hadoop
                 * tar -xvf /opt/hadoop-1.2.1.tar.gz
解壓 hadoop
         config
                 * echo "HADOOP_HOME=/opt/hadoop-1.2.1" >> /etc/profile
                 * source /etc/profile
                 * cd /opt/hadoop-1.2.1
                 * vi conf/hadoop-env.sh
設定hadoop環境的JAVA_HOME
                         edit: export JAVA_HOME=/opt/jdk1.7.0_45
                 * vi conf/core-site.xml
                         edit: <configuration>
<property>
<name>fs.default.name</name>
<value>hdfs://localhost:9000</value>
</property>
</configuration>
HDFS 服務 port: 9000
                 * vi conf/hdfs-site.xml
                         edit: <configuration>
<property>
<name>dfs.replication</name>
<value>1</value>
</property>
</configuration>
檔案副本數量1
                 * vi conf/mapred-site.xml
                         edit: <configuration>
<property>
<name>mapred.job.tracker</name>
<value>localhost:9001</value>
</property>
</configuration>
Mapreduce 服務 port: 9001

                 * cat ~/.ssh/id_dsa.pub >> ~/.ssh/authorized_keys

                 v ssh localhost
