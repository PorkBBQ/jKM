Commands
         essential
                 ls
查看檔案
                         usage: hadoop fs -ls <args>
                 lsr
查看檔案 遍歷目錄
                         usage: hadoop fs -lsr <args>
                 put
複製檔案至HDFS, 使用/不使用CRC檢查
                         usage: hadoop fs -put <localsrc> ... <dst>
                         example
                                 hadoop fs -put localfile /user/hadoop/hadoopfile
                                 hadoop fs -put localfile1 localfile2 /user/hadoop/hadoopdir
                                 hadoop fs -put localfile hdfs://nn.example.com/hadoop/hadoopfile
                                 hadoop fs -put - hdfs://nn.example.com/hadoop/hadoopfile
Reads the input from stdin.
                 get
從HDFS複製檔案, [CRC]檢查檔案
                         usage: hadoop fs -get [-ignorecrc] [-crc] <src> <localdst>
                         example
                                 hadoop fs -get /foo.txt /tmp/foo.txt
                                 hadoop fs -get hdfs://foo.txt /tmp/foo.txt
                 cat
查看檔案內容
                         usage: hadoop fs -cat URI [URI …]
                         example
                                 hadoop fs -cat hdfs://temp.foo.txt
                                 hadoop fs -cat file:///tmp/foo.txt
更改檔案群組, [-R]遍歷目錄
                 tail
列出檔案最後1K資料
                         usage: hadoop fs -tail [-f] URI
                         example
                                 hadoop fs -tail pathname
                 rm
移除空目錄
                         usage: hadoop fs -rm URI [URI …]
                 rmr
移除目錄, 遍歷檔案
                         usage: hadoop fs -rmr URI [URI …]
                 mkdir
建立目錄, 可複數
                         usage: hadoop fs -mkdir <paths>
                 mv
移動檔案, 可複數
                         usage: Usage: hadoop fs -mv URI [URI …] <dest>
                         example
                                 hadoop fs -mv hdfs:///ile1 hdfs://file2 hdfs://file3 hdfs:///dir
         others
                 copyFromLocal
同put, 但來源必須是local
                         usage: hadoop fs -copyFromLocal <localsrc> URI
                 copyToLocal
同get, 但目的地必須是local
                         usage: hadoop fs -copyToLocal [-ignorecrc] [-crc] URI <localdst>
                 cp
在HDFS複製檔案 可有多個來源
                         usage: hadoop fs -cp URI [URI …] <dest>
                         example
                                 hadoop fs -cp /user/hadoop/file1 /user/hadoop/file2
                                 hadoop fs -cp /user/hadoop/file1 /user/hadoop/file2 /user/hadoop/dir
                 du
檔案長度
                         usage: hadoop fs -du URI [URI …]
                 dus
檔案長度summary
                         usage: hadoop fs -dus <args>
                 expunge
Empty the Trash
                         usage: hadoop fs -expunge
                 getmerge
合併檔案, addnl在檔案之間斷行
                         usage: hadoop fs -getmerge <src> <localdst> [addnl]
                 setrep
改變檔案副本量, [-R]遍歷檔案
                         usage: hadoop fs -setrep [-R] <path>
                         example
                                 hadoop fs -setrep -w 3 -R /user/hadoop/dir1
                 stat
傳回路徑狀態
                         usage: hadoop fs -stat URI [URI …]
                 test
檢查檔案 [-e]檔案存在, [-z] 零長度, [-d]是否目錄
                         usage: hadoop fs -test -[ezd] URI
                         example
                                 hadoop fs -test -e filename
                 text
以test format顯示檔案內容, allow zip and TextRecordInputStream
                         usage: hadoop fs -text <src>
                 touchz
建立空檔案
                         usage: hadoop fs -touchz URI [URI …]
                 chgrp
                         usage: hadoop fs -chgrp [-R] GROUP URI [URI …]
                 chmod
更改檔案權限, [-R]遍歷目錄
                         usage: hadoop fs -chmod [-R] <MODE[,MODE]... | OCTALMODE> URI [URI …]
                 chown
更改檔案所有者, [-R]遍歷目錄
                         usage: hadoop fs -chown [-R] [OWNER][:[GROUP]] URI [URI ]
